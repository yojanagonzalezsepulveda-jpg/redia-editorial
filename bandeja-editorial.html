LinkedIn — Página REDIA
Para publicar necesitas crear una app en linkedin.com/developers y obtener un Access Token con permisos de escritura en la página REDIA.
Antes de llenar esto:
1. Ve a linkedin.com/developers → "Create app"
2. Nombre: REDIA Editorial · URL: redia.pro
3. En "Products" agrega: "Share on LinkedIn" y "Marketing Developer Platform"
4. En "Auth" copia el Client ID y Client Secret
5. Genera un Access Token con scope w_member_social
Facebook — Página REDIA
Para publicar necesitas un Page Access Token permanente de la página REDIA en Facebook.
Antes de llenar esto:
1. Ve a developers.facebook.com → "My Apps" → "Create App"
2. Tipo: "Business" · Nombre: REDIA Editorial
3. Agrega producto "Pages API"
4. En Graph API Explorer genera un Page Access Token para la página REDIA
5. Convierte el token a "Long-lived" (dura 60 días)

· Cercano → Entrevistas y casos de éxito
· Editorial → Contenido patrocinado y avisos
